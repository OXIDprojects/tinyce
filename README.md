# TinyCE for OXID-eSales

## Installation

- Download the Module
- Upload the tinyce folder to /modules

### OXID 4.5

- Add this module to your modules list (OXID Backend > System > Modules)

    oxoutput => tinyce/tinyce_oxoutput
